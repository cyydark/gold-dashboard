API 设计（修订版）

fallback 机制移除 · 后端只调用指定源

后端原则

后端只做一件事

  • 用户选什么源，后端就调那个源
  • 成功 → 返回数据
  • 失败 → 返回错误（HTTP 500 + 错误信息）
  • 不重试，不 fallback，不自动切换源

前端负责兜底

  • 轮询失败 → 显示上次缓存数据 + 错误提示
  • 用户可手动切换到其他源
  • localStorage 记住用户选择
接口（不变）
GET /api/realtime/xau/{source}
GET /api/realtime/au/{source}
GET /api/realtime/fx/{source}
GET /api/chart/xau?source=...
GET /api/chart/au?source=...
数据源排序（仅供前端默认值参考，不做自动切换）
XAUUSD: COMEX → Metals(LBMA) → Sina → Binance → Omkar(CME)
AU9999: 东方财富 → Sina → T+D → 沪金期货(AkShare)
USDCNY: yfinance → Sina
排序用于：首次加载时默认选第一个 · 用户手动切换失败时提示可用的下一个源
