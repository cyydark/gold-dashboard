✅ 移除 DB 后收益

  • 无 SQLite 文件，无迁移，无 ORM
  • 无定时写入任务，无 worker
  • 无数据库连接，无 session 管理
  • 部署更简单（纯 stateless API）

⚠ 需确认

  • Alerts/周报等是否依赖 DB？需确认 scope
  • 历史数据（超过72小时的K线）如需保留，也依赖 DB
  • 如果不需要历史数据，则可完全移除 DB
